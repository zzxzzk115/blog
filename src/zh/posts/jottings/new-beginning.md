阿里云、腾讯云的学生优惠马上就都用完了，后续服务器的费用还是不小的。为了节约，我打算还是用回 GitHub Pages。这次，我并不打算使用 Hexo，而是考虑使用具有 Vue 特性的更强大的静态页面生成器：VuePress。我选择 Hope 作为主题，然后慢慢加入一些内容进来。希望这个博客更简约、更清爽，更适合阅读。

为了提高我的英文写作能力，我决定，后续发布的大多数文章都会有中英两个版本。
